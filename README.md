# 2-dimensional Rocket Simulation
Interactive space rocket flight model - physics education project

### Gameplay
![](demo.gif)

### Controls
1: slow down sim_speed
2: increase sim_speed
3: zoom -
4: zoom +
L_ALT: next cam

L_CTRL: throttle down
L_SHIFT: throttle up
ARROW LEFT: turn left
ARROW RIGHT: turn right
ARROW UP: engine on
